# Summarize YouTube Videos from Transcript

## Overview
Generates concise summaries from YouTube transcripts using LLMs.

## What this workflow does
- Automates a repeatable task end‑to‑end in n8n
- Connects third‑party services via APIs and triggers
- Produces a consistent output with minimal manual effort

## Files
-  — n8n workflow export

## Setup
1. Import the workflow into n8n (**Settings → Import from File**).
2. Add credentials for the services used in the workflow.
3. Update any environment variables or API keys referenced.
4. Run a test execution and review outputs.
